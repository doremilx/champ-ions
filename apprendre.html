Installation de Linux

1Téléchargement

Clique sur le bouton pour télécharger la version de Linux NIRD.

2Création de la clé

Insère une clé USB vierge sur laquelle on grave le fichier téléchargé.

3Redémarrage

Garde la clé USB insérée et redémarre ton appareil.

4Ouvrir le menu Boot

Dès le redémarrage, appuie sur la touche (souvent F12) pour ouvir le menu Boot de ton appareil.

5Utilisation de la clé

Dans le menu choisi de démarrer avec l'option USB.
